# Project Memory

## Core
WhatSaid: AI audio transcription app. Space Grotesk headings, Inter body.
Primary hsl(250 75% 55%), accent hsl(170 65% 45%), dark mode supported.
Lovable Cloud for auth/db/storage. AssemblyAI for STT. Lovable AI for post-processing.
Credit model: 1 credit per file. Base GBP prices: £4.99 one-time, £14.99 5-pack, £39.99 20-pack.
Audio deleted immediately after processing. No audio retention.
Paddle as merchant of record (replacing Stripe). Guest pricing fully removed.
