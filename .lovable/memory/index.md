# Project Memory

## Core
WhatSaid: AI audio transcription app. Space Grotesk headings, Inter body.
Primary hsl(245 50% 48%), accent teal (success only), dark mode supported.
Lovable Cloud for auth/db/storage. AssemblyAI for STT. Lovable AI for post-processing.
Credit model: 1 credit = 15min, 2 = 30min, 3 = 45min, 4 = 60min. Guest one-off pricing.
Audio deleted immediately after processing. No audio retention.
Glass effects: navbar only. Solid surfaces everywhere else.
Badge rounded-lg not rounded-full. Buttons rounded-lg. Cards rounded-xl.
